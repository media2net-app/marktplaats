# Service ID Vinden - Directe Instructies

Je bent nu op: `https://railway.com/project/d7cf2713-0d81-4820-b62f-cd732285675c`

## ‚úÖ Stap 1: Klik op de Service

Je ziet een kaart met "marktplaats" en "Online" status.

1. **Klik op die kaart** (de "marktplaats" service)
2. Je gaat nu naar de service pagina

## ‚úÖ Stap 2: Kijk naar de URL

Na het klikken op "marktplaats", wordt de URL:

```
https://railway.com/project/d7cf2713-0d81-4820-b62f-cd732285675c/service/{SERVICE_ID}
```

**Het deel na `/service/` is je Service ID!**

Bijvoorbeeld:
- Als de URL is: `https://railway.com/project/.../service/abc123-def456-ghi789`
- Dan is je Service ID: `abc123-def456-ghi789`

## ‚úÖ Stap 3: Of via Settings

1. **Klik op "marktplaats"** service
2. **Klik op "Settings"** (bovenaan, naast "Architecture", "Observability", "Logs")
3. **Scroll naar "General"** sectie
4. **Zoek "Service ID"** - kopieer deze waarde
